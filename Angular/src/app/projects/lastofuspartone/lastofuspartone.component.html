Last of Us Part I

The Last of Us Part I is an AAA action-adventure game released in September 2022 for the Playstation 5. I was a R&D QA Programmer developing software and tools to improve development and testing workflows.

I worked on many different applications by collaborating with developers of all backgrounds from different departments to solve inefficiencies in the workflow. I enhanced Jira functionality by developing Chrome extension tools, added debug features to the studio's proprietary game engine, and created internal webpages which display all performance stats/notes from an automated daily test.


On top of developing tools, I investigated defects in the game, reported/logged issues with detailed reproduction steps including screenshots/videos using Jira, and thoroughly examined fixes to ensure a quality product.
